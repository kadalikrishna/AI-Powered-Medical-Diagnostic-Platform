# AI-Powered Medical Diagnostic Platform

An AI-integrated web application that allows users to upload medical reports and receive AI-generated diagnostic insights using the Google Gemini API. The platform provides a responsive user interface, secure authentication features, and real-time notifications for better user interaction.

## Features

* Upload and analyze medical reports
* AI-generated insights using Google Gemini API
* User authentication and secure password handling
* Real-time notifications using Socket.io
* Responsive frontend interface
* PostgreSQL database integration
* Frontend deployment on Vercel and backend deployment on Render

## Tech Stack

* React.js
* Node.js
* Express.js
* PostgreSQL
* Socket.io
* Google Gemini API
* Tailwind CSS
* Vercel
* Render
